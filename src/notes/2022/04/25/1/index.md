With JavaScript, how would you compute all sorted permutations of 1 to n elements from a given array?

For example, with this source `[1, 2, 3]` and `n=2`

How would you get this?

```javascript
[[1], [2], [3], [1, 2], [1, 3], [2, 3]];
```
